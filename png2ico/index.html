常见问题

ICO 和 PNG 有什么区别？

ICO 是 Windows 系统原生的图标格式，可以在单个文件中存储多种分辨率的图像，系统会根据显示需求自动选择合适的尺寸。PNG 是通用图片格式，不支持多分辨率存储，但在网页中也常被用作 favicon（需配合 link 标签指定类型）。

网站 favicon 应该用哪个尺寸？

推荐生成包含 16×16、32×32、48×48 的合并 ICO 文件。现代浏览器会根据设备像素比选择合适尺寸。如果只想用一个尺寸，32×32 的兼容性最好。

转换后 ICO 质量会下降吗？

工具提供「高质量（双三次插值）」选项，在缩放时使用插值算法保持图像质量。建议上传尺寸足够大的原始图片（如 512×512 或更大），再生成小尺寸 ICO，以获得最佳缩放效果。

支持哪些输入图片格式？

支持 PNG、JPG/JPEG、WebP、GIF 等常见图片格式。推荐使用带透明背景的 PNG 格式，这样转换出的 ICO 图标在各种系统主题下显示效果最佳。
