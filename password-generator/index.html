免费在线密码生成器

本工具使用浏览器内置的 Web Crypto API 生成加密级随机密码，所有计算在本地完成，密码不会上传至任何服务器，安全可靠。

如何生成强密码？

建议密码长度至少 12 位，同时勾选大写字母、小写字母、数字和特殊符号四种类型，可将密码强度提升至"强"级别。重要账户（如邮箱、银行）推荐使用 16 位以上。

常见问题

这个密码生成器安全吗？
是的。密码完全在您的浏览器本地生成，使用 crypto.getRandomValues 产生真随机数，不联网、不记录，关闭页面后数据即销毁。
什么是易混字符？
指 I l 1 O 0 o 等视觉上容易混淆的字符。如需手动输入密码（如在电视或键盘受限设备上），建议勾选"排除易混字符"。
推荐密码长度是多少？
日常账户建议 12–16 位；金融、邮箱等高价值账户建议 20 位以上；配合密码管理器使用时可生成 32 位以上的超强密码。
